# FSD – Applying JavaScript

## Project Overview

This repository contains basic practice files created to understand how JavaScript is applied in web development.

The project demonstrates how JavaScript can be connected to HTML files and used to add functionality and interactivity to webpages.

It is created as part of Full Stack Development (FSD) learning.


## Objectives

- Learn how to link JavaScript with HTML
- Understand external JavaScript files
- Practice basic JavaScript concepts
- Improve interactivity in webpages
- Strengthen frontend development fundamentals


## Technologies Used

- HTML5
- JavaScript

## File Description

### 1. Document.html

Demonstrates basic HTML structure with JavaScript integration.
Shows how JavaScript interacts with HTML elements.


### 2. Embedding.html

Demonstrates different ways of embedding JavaScript in HTML:
- Inline JavaScript
- Internal JavaScript
- External JavaScript


### 3. submit.html

Demonstrates form submission handling using JavaScript.
Includes basic validation or event handling concepts.


### 4. voterinfo.html

Demonstrates a simple information form or data handling using JavaScript logic.


### 5. script.js

Contains JavaScript code linked externally to HTML files.
Used to perform basic operations and DOM manipulation.


### 6. script1.js

Contains additional JavaScript functions for practice.
Demonstrates event handling or user interaction.


### 7. script2.js

Includes more JavaScript examples for learning purposes.
May include validation, conditions, or simple logic.


## How to Run

1. Clone the repository:

```
git clone <repository-link>
cd FSD-Applying-JS
```

2. Open any HTML file in a web browser.

No additional installation or dependencies are required.
